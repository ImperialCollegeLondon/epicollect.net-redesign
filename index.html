Mobile / Web Application for Smartphone data collection

EpiCollect.net provides a web and mobile app for the generation of forms (questionnaires) and freely hosted project websites for data collection. Data are collected (including GPS and media) using multiple phones and all data can be viewed centrally (using Google Maps / tables / charts).

For simple projects:

  • Drag and Drop form Builder
  • Define a single form for survey
  • Any number of text questions
  • Add a single GPS location + photo
  • View / chart with Maps
  • Download all data
  • Greater Form Validation
  • Define form logic (jump Q's)
  • Upload ecml form definitions
  • Android & iOS app
