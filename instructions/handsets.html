Handsets & Platforms supported

The EpiCollect mobile applications are currently available for the Android and iOS Operating Systems. Any phone model that currently runs Android should and iOS should run one of the versions of EpiCollect. However, we recommend that at least Android version 1.6 is running on your phone.

See below table for details

EpiCollect Version Server Android iOS
EpiCollect Google App Engine Android 1.6+ iOS 7+
EpiCollect+ Beta Imperial College Server or personal server Android 1.6+ (Old App) , Android 4.1+ (New App) iOS 7+

Please be aware we are currently actively developing only for Epicollect+ Beta as the other versions will be deprecated soon
